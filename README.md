# REST API With Flask & SQL Alchemy

<img src="https://www.ncrts.com/public/images/webapp-banner.jpg">
> Form Database API using Python Flask, SQL Alchemy and Marshmallow

## How to run it

``` bash
# Run Server (http://localhost:5000)
python app.py
```

## Endpoints

* GET     /all
* GET     /create
* POST    /create

## Technologies

* HTML5
* CSS
* Python
* SQLAlchemy
* Flask
* Marshmallow
